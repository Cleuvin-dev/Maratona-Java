# Maratona-Java
Todo o percurso de aprendizado do Java até tópicos avançados no formato de códigos documentados como se fossem um livro de bolso, 
um guia prático para consulta, caso houver alguma duvida de algo a ser usado.
<br><b><i>Lembrado que existem comentarios nos codigos referente ao aprendizado das aulas e para futuras consulta, nao seguindo o padrão Clean Code, pois será totalmente didático.</b></i>

The entire Java learning path to advanced topics in code format documented as if they were a pocket book, 
a practical guide for consultation, if there is any doubt about something to be used.
